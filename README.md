Tokeruは「**1日の作業パフォーマンスを最大化するアプリ**」です。 (macOS 11.0以降)。  
`⇧ ⌘ ,`キーでSpotlightやAlfredのようなTodoウィンドウが開きます。  
ショートカットにも対応しており、作成、チェック、ウィンドウを閉じる操作はキーボードのみで行えます。  

まだTodo機能しかありませんが、今後は時間をうまく使うための機能や思考の整理のための機能を実装予定です。
